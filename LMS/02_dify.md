# Difyってなんだろう

このパートでは、今回メインで触っていくDifyについて「どんなことができるサービスなのか」見ていきましょう。

## 1. Difyについて

Dify(ディフィー)は、チャットボットや自動化の仕組みをノーコードやローコードで作ることができるAI開発プラットフォームです。 

- 2025年5月：公式が呼び名を「**ディフィー**」と発表しました。
- 現時点（2026年3月）のクラウド版無料枠では、200メッセージまで利用可能で、アプリは5個まで作成できます。  
参考：[Dify](https://dify.ai/jp/pricing)｜プランの比較  

### 1-1. Difyでできること

Difyを使うと、生成AIやLLM単体では難しいことも自分で形にしやすいです。  
`情報を参照する → 文章にまとめる → メールで送る`といった一連の業務の流れをAIに任せた自動化の仕組みも作ることができます。  

また、周りの人に使ってもらうための機能も充実しています。 

- **作ったアプリを公開できる**：  
Webサイトに埋め込んだり、URLを共有したりできます。
- **他の人にも簡単に使ってもらえる**：  
アカウント登録なしで、作成したAIを利用してもらえます。

### 1-2. アプリケーションの種類

アプリケーションの種類は5つあるため、作るものに合わせて選びましょう。  

※ 資料では「**チャットフロー**」と「**ワークフロー**」を扱ったハンズオンを取り扱っています。

> <img src="https://i.gyazo.com/71f97e9b1fd40a0c1bfa1c8e382e7124.png" width="450px" />

| アプリタイプ | 特徴 |
|---|---|
| **チャットフロー** | 会話の流れをあらかじめ設計し、質問内容や状況に応じて分岐する対話ができる |
| **ワークフロー** | 人が決めた手順どおりに、処理や作業を順番に自動実行する仕組み |
| チャットボット | 質問に対して会話形式で答える。主に案内やFAQなどのシンプルな対話に向いている |
| エージェント | 目的に応じて何をするかを考え、調査や処理を組み合わせて動く仕組み |
| テキストジェネレーター | 入力に対して、1回きりの文章や文章案を生成する仕組み |

## 2. オンプレミス環境で構築できる

Difyは、**オンプレミス環境（自社管理のサーバーやPC）に構築して利用できる**点が大きな特徴です。

日本の大企業では、生成AIを利用する際に**外部のサーバーと切り離した環境で使いたい**というニーズが多くあります。  
完全なクラウドサービスに限定されず、オンプレミス利用も選択できる点は、Difyの大きな強みです。

Difyと似た立ち位置のクラウドサービスに、miiboやCozeがありますが、**クラウド版とオンプレミス版を選べる**のはDifyのみです。

## 3. OSSでの開発

Difyは、OSS（オープンソースソフトウェア）としてソースコードが公開されています。 

> OSSとは？  
> ソースコードが公開されており、誰でも中身を確認できるソフトウェアのこと。 

### 【OSSの特徴】

- ソースコードを自由に閲覧できる  
- 条件の範囲内で利用・改変・再配布ができる  
- 多くの開発者が参加し、継続的に改善される

OSSであることは、「無料で使える」だけでなく**中身が見えることによる透明性や、長期的な安心感**につながります。

Difyには多くの開発者が関わっており、開発が活発に進みどんどんより良いサービスへと成長しています。  
コミュニティもあり、日々使い方や活用事例も共有されています。
